# Security Policy

## Scope

Open Tooling CRM is designed as a local-only, self-hosted application. It does not include authentication or authorization — it is expected to run on trusted networks or behind a reverse proxy that handles access control.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately via GitHub's security advisory feature rather than opening a public issue.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |
| < 1.0   | No        |
